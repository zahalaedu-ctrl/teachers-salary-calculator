# Teachers Salary Calculator

מחשבון שכר ידידותי לעובדי הוראה בישראל.

## מטרת הפרויקט

פיתוח מחשבון שכר פשוט, ברור ואמין לעובדי הוראה, בדגש על אופק חדש בשלב הראשון.
המערכת תאפשר למורה להזין נתונים מוכרים ופשוטים — כגון שעות פרונטליות, דרגה, ותק, השכלה וגמולי תפקיד — ולקבל אומדן שכר ברור ומוסבר.

## עקרונות מרכזיים

- המשתמש אינו נדרש לדעת לחשב אחוז משרה.
- המערכת שואלת על שעות פרונטליות ומחשבת את ההיקף מאחורי הקלעים.
- תומכים תחילה במקרים השכיחים בלבד.
- חריגים יוצגו כביאורים או כאפשרות מתקדמת.
- מנוע החישוב יהיה מופרד מהממשק.

## שלב ראשון

MVP לאופק חדש בבית ספר יסודי.
